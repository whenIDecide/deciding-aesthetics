Deciding Aesthetics

Development Solutions

Deciding Aesthetics is a project created by Matthew Farr as a home for his professional work as a developer with a focus on design. The ethos of Deciding Aesthetics is two fold:

  • First that if you pursue code that looks good, you end up with more funtional code as a by-product. (Besides, who doesn't like some good clean code?)
  • Second that whenever possible the function of code should be planned with the desired aesthetic of the eventual product in mind. A singular vison for aesthetic and function makes the work in both categories more efficient and successful.
